Offline & Sincronização
Persistência local (IndexedDB), instalação como app e sincronização com o servidor
● Online
💾 Dados locais (offline)
Tudo o que você cadastra fica salvo neste dispositivo e funciona sem internet. Os dados sobrevivem ao recarregar a página.
📲 Instalar como aplicativo (PWA)
Instale no celular ou tablet para abrir em tela cheia, com ícone próprio e acesso offline.
ℹ️
A instalação exige que o sistema esteja hospedado em um endereço HTTPS (não funciona abrindo o arquivo direto do computador). Veja o passo a passo no manual.
☁️ Sincronização com o servidor (Neon / API)
Informe o endereço da API backend que conversa com o banco Neon. Por segurança, o sistema nunca usa a string do banco diretamente — quem guarda a senha é o backend.
